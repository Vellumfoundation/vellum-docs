## How to verify a link is official

- Check this page first.
- Cross-check against this documentation if you reach a Vellum surface from another source.
- Be suspicious of:
  - Lookalike domains.
  - Pages with values that do not match this documentation.
  - Anyone offering "free ETH on Vellum mainnet."
  - Anyone asking for a seed phrase or private key.

{% hint style="danger" %}
Vellum infrastructure will never ask for a seed phrase or private key.
{% endhint %}

## Reporting impersonation

If you encounter a phishing site, fake bridge, or impersonator:

- Do not connect a wallet.
- Do not interact with it beyond what is needed to confirm the issue.
- Report it through the support contact above.
